# XO Enchanted

## Project Overview
XO Enchanted is an engaging two-player game that combines classic tic-tac-toe elements with enchanting graphics and sound effects. It challenges players to think strategically while providing a fun user experience.

## Features
- Two player mode (local)
- Interactive graphics
- Sound effects
- User-friendly interface

## Game Rules
1. The game is played on a 5x5 grid.
2. Players take turns placing their symbols (X or O) on the grid.
3. The first player to align five of their symbols vertically, horizontally, or diagonally wins the game.
4. If the grid is filled without any player winning, the game ends in a draw.
