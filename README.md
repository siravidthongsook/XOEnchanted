## Game Overview
XOEnchanted is an immersive and engaging game that combines strategic gameplay with enchanting graphics. Players dive into a magical realm where they can challenge their skills against a series of AI or human opponents.

## Game Features
- **Special Skills System**: Each character in XOEnchanted possesses unique skills that can turn the tide of the game. Players can strategically activate these skills to gain advantages over their opponents.

## Installation Instructions
1. **Clone the Repository**:  
   ```bash  
   git clone https://github.com/siravidthongsook/XOEnchanted.git  
   ```  
2. **Navigate to the Project Directory**:  
   ```bash  
   cd XOEnchanted  
   ```  
3. **Install Dependencies**:  
   Ensure that you have Gradle installed. Run the following command:  
   ```bash  
   gradle build  
   ```

## Building and Running with Gradle
To build and run the project, use the following command:  
```bash  
gradle run  
```
## Game Rules and Mechanics
- Players take turns placing their marks on the board. The goal is to align three of your marks in a row—vertically, horizontally, or diagonally.  
- Special skills can be activated once certain conditions are met, giving players an edge in their strategy.  
- The game ends when a player wins by aligning three marks or when the board is full leading to a draw.
